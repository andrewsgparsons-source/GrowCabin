# Grow Cabin — Investor pack (checklist)

This document defines the assets we should maintain for investor conversations.

## Core pack (must have)
1) **Deck (8–12 slides)**
2) **One-pager** (PDF)
3) **Prototype demo assets** (photos + short videos)
4) **Cost & pricing model** (BOM + assumptions)
5) **Roadmap** (milestones + timing)

## Strong add-ons
- Interview notes and segmentation results
- Waitlist metrics
- KPI table (success rate, days-to-harvest, time spent/day, running cost)
- Competitive landscape (1 slide)
- Risk register + mitigations (mold/contamination, safety, reliability)

## Draft deck structure (recommended)
1. Title
2. Problem
3. Why now
4. Solution
5. Product
6. Segment & use case
7. Go-to-market
8. Traction / validation
9. Unit economics
10. Roadmap
11. Team
12. Ask

## Open items
- Decide the investor lane and initial target cheque size
- Decide price band and MVP scope for the first deck
