# Product design — environmental controls

## Control loops
- Humidity control (hysteresis first; PID later if needed)
- FAE scheduling; optional CO₂ sensor
- Temperature (optional heater)
- Lighting schedule for greens

## Two tiers
- **Essential (plug-and-play)**
- **Enhanced (IoT / logging / remote monitoring)**
