# Grow Cabin — 30s Promo Script + Shot List (v0)

_Last updated: 2026-02-11_

Purpose: first executable creative draft for a short promo video using hybrid real footage + AI enhancement.

## 1) Positioning angle
"Reliable home harvests, without the guesswork."

## 2) 30-second script (voiceover)
**0–4s**
"Growing at home should be rewarding — not random."

**4–10s**
"Grow Cabin gives you controlled conditions for mushrooms and greens, with recipe-guided stages."

**10–18s**
"Sensors monitor the environment. Smart control keeps humidity and airflow where they need to be."

**18–24s**
"Less trial-and-error. More repeatable harvests."

**24–30s**
"Grow Cabin. Clean, calm, and consistent home growing."

## 3) Shot list (30s)
1. **Hero open** (0–4s)
   - Cinematic product silhouette + soft mist/light
   - Text: "Home growing, made reliable"

2. **Interaction shot** (4–8s)
   - Hand opening chamber/door, checking trays/blocks

3. **Environment close-ups** (8–14s)
   - Sensor readouts / airflow / humidity plume visuals
   - Overlay icons: temp, RH, airflow

4. **Process montage** (14–22s)
   - Stage cards/recipe UI + short growth progression shots

5. **Outcome shots** (22–28s)
   - Healthy mushrooms + greens, clean setup, easy daily routine

6. **End card** (28–30s)
   - Logo + tagline + call-to-action

## 4) Production notes
- Style: grounded, premium, not over-hype
- Preferred format output set:
  - 16:9 (YouTube/site)
  - 9:16 (Reels/TikTok)
- Music: warm, optimistic, low distraction
- SFX: subtle fan/humidifier ambience for credibility

- "Join the pilot"
- "See the prototype"
- "Get early access"
