# KPIs (starter set)

These KPIs are designed to be understandable by customers *and* useful for investors.

## Grow success / reliability
- **Success rate**: % grows that complete without mold/contamination/failure
- **Time-to-first-harvest** (days)
- **Repeatability**: variance across runs using the same recipe

## Output
- **Yield** per tray/block
- **Quality score** (taste/texture; simple rubric)

## Effort
- **Minutes per day** user effort
- **Number of interventions** per grow cycle

## Cost
- **kWh per harvest**
- consumables cost per cycle

## Support
- % grows needing support
- time-to-resolution
